# xrwvm-fullstack_developer_capstone

## Project Name
fullstack_developer_capstone

## Description
This project is a full stack web application built using Django and React. It allows users to view dealerships, submit reviews, and analyze sentiments.

## Features
- User login and logout
- Dealer listing and filtering
- Review submission
- Sentiment analysis
- Admin panel

## Technologies Used
- Django
- React
- Bootstrap
- REST APIs
